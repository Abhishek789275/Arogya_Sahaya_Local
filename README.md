# Arogya-Sahaya Local

Arogya-Sahaya Local is an Android healthcare assistance application developed using Kotlin and Android Studio.  
The application helps rural and elderly patients manage medicines, appointment reminders, and healthcare follow-ups in a simple and user-friendly way.

---

## 📱 Features

- 💊 Medicine Reminder System
- 📅 Doctor Appointment Reminder
- 🏥 Basic Health Record Management
- 🔔 Notification Alerts
- 🌐 Simple and Easy UI
- 📱 Android-Based Healthcare Support

---

## 🚀 Technologies Used

- Kotlin
- Android Studio
- Jetpack Components
- Room Database
- XML Layouts
- Gradle

---

## ❓ Problem Statement

In many rural areas, elderly patients often forget medicines and healthcare follow-ups because managing multiple medications and remembering hospital visits is difficult.

Arogya-Sahaya Local helps users stay updated with reminders and simple healthcare management features.

---

## 🎯 Objectives

- Improve medication adherence
- Help patients remember appointments
- Provide simple healthcare assistance
- Support rural healthcare awareness
- Build an easy-to-use Android application

## ⚙️ Installation Steps

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Abhishek789275/Arogya_Sahaya_Local.git
```

### 2️⃣ Open in Android Studio

- Open Android Studio
- Click on **Open Project**
- Select the cloned folder

### 3️⃣ Sync Gradle

Wait for Gradle build and dependency sync.

### 4️⃣ Run Application

- Connect Android device or start emulator
- Click ▶ Run
